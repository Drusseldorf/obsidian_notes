# Клавиатуры

Reply и Inline клавиатуры

```python
	# reply - отрисовываются ниже поля, где пользователь вводит текст сообщения. При нажатии - текст отправляется в чат. Так же могут запрашивать локацию, контакты, другие данные пользователяю

	# inline - могут быть отосланы как сообщение пользователю, отрисовываются в сообщениях, при нажатии - могут слать колбэк дату, которая будет отлавливаться хэндлерами. Можно открывать ссылки или приложения, встроенные в ТГ

	# ReplyKeyboard:

		from aiogram.types import ReplyKeyboardMarkup, KeyboardButton, InlineKeyboardMarkup, InlineKeyboardButton

		main_1 = ReplyKeyboardMarkup(keyboard=[
			[KeyboardButton(text='Каталог')],
			[KeyboardButton(text='Корзина'), KeyboardButton(text='Контакты')]
		],
			resize_keyboard=True,
			input_field_placeholder='Выберите пункт меню')

	input_field_placeholder - будет написано в строке для ввода текста у пользователя
	resize_keyboard - по дефолту кнопки слишком большие, этот аргумент делает их поменьше

	тут нужно обратить внимание, что в keyboard передаются списки списков.
	Например тут в примере,первый список - состоит из одного элемента, а значит у нас в клавиатуре ПЕРВЫЙ РЯД будет из одной кнопки
	В примере, второй список - состоит из двух элементов, а значит у нас в клавиатуре ВТОРОЙ РЯД будет из двух кнопок

	Импортируем клавиатуру main_1 в keyboards и передаем в хэндлер
	Можно ответить клавиатурой:

		import app.keyboards as kb

		@router.message(CommandStart())
		async def cmd_start(message: Message):
			await message.answer('Приветик', reply_markup=kb.main)

	InlineKeyboard:

		main = InlineKeyboardMarkup(inline_keyboard=[
			[InlineKeyboardButton(text='Каталог', callback_data='catalog')],
			[InlineKeyboardButton(text='Корзина', callback_data='cart'), InlineKeyboardButton(text='Контакты', callback_data='contacts')]
		])

	Тут аналогично ReplyKeyboard расположение кнопок будет таким же как расположили элементы в списках

	Кнопки обязательно должны содержать text и еще чтото, например url или callback_data

	Одно сообщение может содержать только одну клавиатуру

	Builder - помогает создавать клавиатуру динамично, по набору данных:

		from aiogram.utils.keyboard import InlineKeyboardBuilder

		cars = ['BMW', 'Audi', 'Tesla']

		async def inline_cars():
			keybord = InlineKeyboardBuilder()
			for car in cars:
				keybord.add(InlineKeyboardButton(text=car, url='<https://example.org>'))
			return keybord.adjust(2).as_markup()

		то есть создали объект клавиатуры, наполнили его кнопками на основе каких то данных и вернули эту клавиатуру. Обязательно через асинх функцию

		.adjust(2) - отредактировать размер, указываем сколько кнопок будет в одном ряду
		.as_markup() - собственно, возвращает клавиатуру

	Как эту функцию передаем в хэндлере (часть кода, где передаем аргумент клавиатуры) -  Обязательно await:

		..., reply_markup=await kb.inline_cars(), ...
```
